# 💼 SK Job Portal

A modern Job Portal Web Application built using Spring Boot, JSP, Maven, and Bootstrap. This application allows users to view job opportunities and add new job postings through a simple and responsive interface.

---

## 🚀 Features

* 🏠 Home Page
* 📋 View All Job Listings
* ➕ Add New Job Posts
* 💼 View Job Details
* 🎨 Responsive UI with Bootstrap
* 🔧 Spring Boot MVC Architecture
* 📦 Maven Project Structure

---

## 🛠️ Tech Stack

### Backend

* Java 21
* Spring Boot
* Spring MVC

### Frontend

* JSP
* HTML5
* CSS3
* Bootstrap 5

### Build Tool

* Maven

### IDE

* IntelliJ IDEA

### Version Control

* Git & GitHub

---


## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Sathish292004/SpringWeb_JobApplication.git
```

### Navigate to Project

```bash
cd SpringWeb_JobApplication
```

### Run Application

```bash
mvn spring-boot:run
```

Application will start on:

```text
http://localhost:8080
```
